# Birthday Countdown

Minimalistic birthday countdown website.

![Mobile](https://github.com/user-attachments/assets/608808f3-b9d7-44b3-820e-723f3967b3f2)

## Setup

1. Edit `config.js` with your details
2. Deploy to any static hosting (GitHub Pages, Vercel, Netlify)
3. Done

## Configuration

Open `config.js` and edit:
- `name`: Person's name
- `birthYear`: Year of birth
- `birthMonth`: Birth month (1-12)
- `birthDay`: Birth day (1-31)
- `birthdayMessage`: Custom birthday message

## Deploy

**GitHub Pages:** Push to repo → Settings → Pages → Deploy from main branch

**Vercel/Netlify:** Connect repo → Auto-deploy
